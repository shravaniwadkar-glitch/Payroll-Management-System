Payroll Management System

Features
- Add employee
- Calculate salary
- Tax slabs
- File handling
- JDBC database integration

Technologies
- Java
- MySQL
- JDBC

Database Setup

1. Open MySQL
2. Run the database.sql file
3. It will create the payroll database and employees table

How to run the project

1. Clone the repository
2. Import the project into your IDE
3. Connect MySQL database
4. Run PayrollManagementSystem.java
